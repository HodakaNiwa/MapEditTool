 ツール概要
	制作環境         :  Windows10/DirectXVer9.0
	制作期間         :  H30年12月〜H31年1月(約２ヶ月)
	制作人数         :  1人
	使用言語         :  C++
	使用ソフト       :  Visual Studio2015
 
 主な操作方法
	****  マウス操作  ****
		視点操作                    :  [ 左クリック ]しながら動かす
		注視点操作                  :  [ 右クリック ]しながら動かす
		カメラを動かす              :  [ 左右同時クリック ]しながら動かす
		配置オブジェクトを動かす    :  [ 中クリック ]しながら動かす
		
	****  キーボード操作  ****
		オブジェクト配置            :  [ ENTER ]キー
		配置オブジェクトの移動      :  [ W ] [ A ] [ S ] [ D ]キー
		配置オブジェクトの高さ変更  :  [ SHIFT ](左側)キーを押しながら[ W ] [ S ]キー
